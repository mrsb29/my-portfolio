Web Design

Project Info:

Oped Academy is an innovative online learning platform designed for students from IGCSE, CBSE, IB, and ICSE boards. It offers live virtual classes, study materials, video lessons, and free career guidance programs. The platform emphasizes student well-being with free yoga sessions and mental health initiatives, alongside regular weekly evaluations through tests.

Project Details:
  • Project Name:Oped Academy - (Online Education Platform)
  • URL:www.opedacademy.org
Key Features:

    Live Virtual Classes, Video Lessons, Study Materials, Weekly Evaluations, Doubt Solving,Mental Health Support,Career Guidance Programs,Parent-Teacher Communication.

Technologies Used:
  • Back-End:

    Laravel for the web framework, ensuring fast, scalable, and secure development.

  • Front-End:

    HTML5, CSS3, Bootstrap 5, and JavaScript to ensure a responsive and clean user interface.

  • Databases:

    MySQL, Advanced SQL Queries

  • Hosting & Deployment:

    Hosted on AWS EC2 with Cloudflare for DNS management and security.
